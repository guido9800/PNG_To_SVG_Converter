Conversion Modes

Photo to line art

Turns color PNGs and photos into high-contrast black line art by converting to grayscale, boosting contrast, detecting image edges, thickening the detected lines, cleaning small artifacts, and tracing those lines as SVG paths.

Best for: photo-style engraving, portraits, product images, and full-color artwork that needs to become mark/no-mark black line art.

Tradeoff: it creates line art, not shaded halftone engraving. Tune Photo contrast and Edge sensitivity before adjusting path smoothing.

Smooth black/white trace

Converts the PNG into a black/white mask, traces the outer contours, simplifies the paths, and optionally rounds the corners. This produces smoother laser-ready vector boundaries than pixel-block tracing.

Best for: two-color galvo engraving artwork, logos, silhouettes, text, and clean mark/no-mark graphics.

Tradeoff: it intentionally reduces the artwork to solid black vector shapes, so use another mode for full-color art.

Optimized vector trace

Converts the PNG pixels into actual SVG vector shapes. The app reduces the image into a color palette, groups same-color pixel runs, and writes those groups as filled paths.

Best for: galvo laser engraving, black and white artwork, logos, high-contrast graphics, and files that need editable vector geometry.

Tradeoff: very detailed photos can create large SVGs because every color region becomes vector geometry.

Exact embedded SVG

Wraps the PNG inside an SVG file without tracing it into editable vector paths. It preserves the original raster image appearance as closely as possible.

Best for: preserving exact visual appearance, previewing DPI and physical sizing, or using software that accepts raster content inside SVG.

Tradeoff: it is still a PNG inside an SVG container, so it is not true vector geometry for cutting or path-based engraving.

Laser engraving recommendation

Use Photo to line art for color photos, and Smooth black/white trace for clean two-color logos. Keep Output DPI between 254 and 300, then tune the mode-specific threshold or edge controls.
